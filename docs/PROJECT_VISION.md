# Stellaris Strategic Intelligence Project
### Rule 0
Strategic Nexus must not do anything illegal or against OpenAI rules.

This is a hard no-go rule.
If a proposed path could violate law, third-party rights, platform terms, OpenAI policy, or safe-use boundaries, stop and choose a safer architecture.

### Rule 1
Vanilla AI stays in control of tactics.

### Rule 2
The external AI layer only influences strategy.

### Rule 3
The daemon must NEVER damage saves.

### Rule 4
The game must continue functioning if the daemon crashes.

### Rule 5
Ironman compatibility is not required for v0.x.
Do not optimize v0.x for achievements.
Non-Ironman runtime integration is acceptable when bounded and safe.

### Rule 6
No invasive manipulation.
No raw LLM command execution.
No direct tactical control.
No fleet control.
No arbitrary command strings from daemon.

### Rule 7
The daemon is NOT the source of truth.
The game and save files remain the source of truth.

### Rule 8
Every empire attempts to maximize its own success.
The system is NOT anti-player.

### Rule 9
Emergent social intelligence is more important than raw optimization.

### Rule 10
Behavior consistency is critical.
Civilizations must feel psychologically coherent.

### Rule 11
Bridge or daemon failure must not break the game.
Fallback scripted behavior must continue without daemon or bridge.

---

## Philosophy

The project focuses on:
- strategic reasoning
- personality
- memory
- geopolitics
- reputation
- emergent diplomacy
- balance-of-power behavior
- long-term adaptation

The project does NOT focus on:
- replacing tactical AI
- perfect economic optimization
- realtime micro-management
- AGI research
- cloud infrastructure

---

## Long-Term Vision

The galaxy develops:
- collective memory
- historical trauma
- reputations
- political blocs
- fear of hegemonies
- opportunistic coalitions
- civilization identities

The result should feel like:
- a living galactic society
- not isolated scripted AI empires
