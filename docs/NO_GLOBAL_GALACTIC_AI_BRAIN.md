# Strategic Nexus - No Global Galactic AI Brain

## Core Rule

Strategic Nexus must NEVER use a single global strategic intelligence context.

Rejected architecture:

global galaxy context
-> centralized reasoning
-> shared hidden information
-> omniscient strategic AI

This architecture is forbidden.

---

# Required Architecture

Strategic reasoning must be empire-scoped.

Each empire reasons only from:
- its own observations
- its own intel
- its own memory
- its own ideology
- its own historical experience
- legitimate diplomatic information

No empire may access hidden strategic knowledge from another empire.

---

# Unknown Empires

Unknown empires must not exist inside strategic reasoning.

If an empire has not:
- established communications
- gained sensor visibility
- gained sufficient intel
- completed first contact
- learned of existence through diplomacy

then that empire must not appear in strategic reasoning contexts.

Correct behavior:
- isolated civilizations remain strategically isolated
- empires may underestimate the galaxy
- regional power structures emerge naturally
- late discovery of hegemonic powers becomes possible

Forbidden behavior:
- automatic galactic awareness
- instant anti-player balancing
- perfect galaxy threat evaluation

---

# Forbidden Systems

The following are forbidden:

- global LLM memory
- global strategic planner
- shared hidden inference
- cross-empire hidden knowledge propagation
- centralized omniscient reasoning
- perfect galactic awareness

---

# Allowed Systems

Allowed:
- independent empire reasoning
- alliance information sharing if supported by game systems
- federation information sharing
- sensor/intel based visibility
- espionage-derived knowledge
- inferred strategic conclusions
- imperfect information
- uncertainty
- outdated beliefs

---

# Design Philosophy

Strategic Nexus must simulate:
- civilizations
- institutions
- strategic cultures
- paranoia
- imperfect intelligence
- historical memory

Strategic Nexus must NOT simulate:
- a godlike galaxy optimizer
- omniscient solver AI
- hidden save-reading intelligence

---

# Anti-Cheat Principle

The player must feel:

"the AI learned about me"

not:

"the AI illegally knows everything."
